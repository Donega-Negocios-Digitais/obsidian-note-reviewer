# 12-01: Apple-Style Design System Components

## Goal
Design and implement Apple-style design system components with consistent styling.

## Requirements
- DSGN-01: Interface exhibits minimalist Apple/macOS-style design aesthetic

## Success Criteria
1. Consistent spacing using 8px grid system
2. SF Pro-like typography (system fonts)
3. Rounded corners (12px-16px for cards, 8px for buttons)
4. Subtle shadows and borders
5. Smooth transitions and animations

## Design Tokens

### Spacing (8px grid)
```typescript
spacing: {
  xs: '4px',    // 0.5rem
  sm: '8px',    // 1rem
  md: '16px',   // 2rem
  lg: '24px',   // 3rem
  xl: '32px',   // 4rem
  '2xl': '48px', // 6rem
  '3xl': '64px', // 8rem
}
```

### Typography
```typescript
fonts: {
  sans: '-apple-system, BlinkMacSystemFont, "SF Pro Display", "Segoe UI", Roboto, sans-serif',
  mono: '"SF Mono", Monaco, "Cascadia Code", "Courier New", monospace',
}

fontSizes: {
  xs: '0.75rem',    // 12px
  sm: '0.875rem',   // 14px
  base: '1rem',     // 16px
  lg: '1.125rem',   // 18px
  xl: '1.25rem',    // 20px
  '2xl': '1.5rem',  // 24px
  '3xl': '2rem',    // 32px
}
```

### Border Radius
```typescript
radius: {
  sm: '6px',
  md: '8px',
  lg: '12px',
  xl: '16px',
  '2xl': '24px',
  full: '9999px',
}
```

### Shadows
```typescript
shadows: {
  sm: '0 1px 2px 0 rgb(0 0 0 / 0.05)',
  md: '0 4px 6px -1px rgb(0 0 0 / 0.1)',
  lg: '0 10px 15px -3px rgb(0 0 0 / 0.1)',
  xl: '0 20px 25px -5px rgb(0 0 0 / 0.1)',
}
```

## Components to Create

### Button Component
- Variants: primary, secondary, ghost, danger
- Sizes: sm, md, lg
- States: default, hover, active, disabled

### Card Component
- Elevated and flat variants
- Optional padding and border
- Hover effect

### Input Component
- Text, email, password, textarea
- Focus states
- Error states
- Label and helper text

### Badge Component
- Solid and outline variants
- Colors: gray, blue, green, yellow, red

### Divider Component
- Horizontal and vertical
- With/without label

## Implementation

### Create Design Tokens File
```typescript
// packages/ui/src/tokens/index.ts
export const tokens = { ... }
```

### Update Tailwind Config
```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      spacing: tokens.spacing,
      borderRadius: tokens.radius,
      boxShadow: tokens.shadows,
    },
  },
}
```

## Apple Design Principles
1. **Clarity**: Clear visual hierarchy
2. **Deference**: Content over chrome
3. **Depth**: Subtle shadows and layers
4. **Minimalism**: Remove unnecessary elements
5. **Consistency**: Predictable patterns

## Notes
- Use system fonts for native feel
- Avoid custom fonts (performance)
- Keep animations subtle (200-300ms)
- Use CSS transforms for smooth animations
