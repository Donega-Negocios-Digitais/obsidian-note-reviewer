07-01: Implement Freemium Tier System with Collaborator Limits

## Goal
Build a tier system that limits free users to individual use while enabling unlimited collaborators for paid subscribers.

## Requirements
- MONY-01: Free tier limits usage to individual (no collaborators)
- MONY-02: Paid tier enables unlimited collaborators with subscription billing

## Success Criteria
1. Free tier users cannot add collaborators to shared documents
2. Paid tier users can add unlimited collaborators
3. Tier status is persisted and checked across sessions
4. Clear upgrade prompts shown to free users when attempting collaboration features

## Implementation Plan

### 1. Create Tier Types
```typescript
// packages/collaboration/src/types/tier.ts
export type SubscriptionTier = 'free' | 'pro';

export interface TierLimits {
  maxCollaborators: number;
  maxSharedDocuments: number;
  hasAdvancedFeatures: boolean;
}

export const TIER_LIMITS: Record<SubscriptionTier, TierLimits> = {
  free: {
    maxCollaborators: 0,
    maxSharedDocuments: 0,
    hasAdvancedFeatures: false,
  },
  pro: {
    maxCollaborators: -1, // unlimited
    maxSharedDocuments: -1, // unlimited
    hasAdvancedFeatures: true,
  },
};
```

### 2. Create useSubscription Hook
